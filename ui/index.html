GENERIC OS

Generic Android Device Profile

Standard configuration suitable for most Android phones (Google Pixel, Motorola, Asus, etc.) connected over USB or Wi-Fi.

⚙️ Standard Settings Applied:

  • 🚀 60 FPS Target Mirroring Frame Rate
  • 🔋 Default Device Power Save Controls
  • 🔊 Standard Audio Routing Alignment (Opus / 128k)
  • ⌨️ Physical keyboard simulation (UHID) enabled
💡

Select another profile in the left menu to view phone-specific optimizations such as the Nothing Glyph Interface or Xiaomi Security bypasses.

NOTHING OS

Nothing Phone Glyph Interface & 120Hz

Nothing OS utilizes unique Glyph LEDs on the back of the device and requires special settings to bypass ambient light delays.

💡 120Hz High-Refresh Optimization:

Nothing Phone supports true 120Hz scrolling. Applying this profile automatically configures the dashboard mirror to request 120 FPS target output limit.

💡 Glyph LED Hardware Controls

MIUI / HYPEROS

Xiaomi Input Quirks

Xiaomi, Redmi, and Poco devices block remote typing on lock screens and system dialogs by default due to strict system overlay restrictions.

⚠️ Crucial Requirement:

  • Navigate to: Settings > Developer Options
  • Enable: USB Debugging (Security Settings)
  • Otherwise, simulated keyboard input and Touch ID unlock requests will be silently blocked.
SAMSUNG KNOX

Samsung Knox Security

Samsung Galaxy devices use Knox Security. Pin unlock requires entering an explicit 'Enter/OK' key after simulated typing.

⚙️ Optimization:

  • The Samsung profile automatically appends the Enter keyevent keycode after typing the backup PIN during biometric unlock sequences.
  • Bypasses default screen dim delays by pulsing virtual cursor interactions.
ONEPLUS / OPPO

OnePlus Custom Controls

OnePlus OS aggressively kills background ADB processes to save battery power.

⚙️ Optimization:

  • Enables a persistent ADB keep-alive daemon that periodically pings the phone every 30 seconds to prevent auto-disconnection.
  • Maintains low latency during long monitoring sessions.
